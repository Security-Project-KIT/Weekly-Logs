# 今日の内容の要約
今日はプロトコルの種類などの１年生の時に習ったものだけではなく、DNSの種類などの新しい知識も得ることが出来た。例えば、デバイスがIPを取得する時のプロトコルはDNSで通信方式はUDPである。また、HTMLを取得する時にはプロトコルはHTTPやHTTPSで通信方式は基本的にUDPだけとUDP+QUICである。そして、DNSにはいろいろな種類がありroot DNS やtld DNS, 権威DNSなどがある。
# JavaScriptとTypeScriptの違い
C言語やJava等の言語では`int`や`double`等のデータ型で引数の種類が分かれているが、JavaScriptでは`const`と`let`だけで引数の型を分ける。TypeScriptは他の言語にあるデータ型を分ける機能を追加したJavaScriptのフレームワークである。
